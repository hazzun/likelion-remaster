## Introduce
### `도움이 필요한 고령자들을 위한 서비스` <br>
문제점 인식
- 도움이 필요한 고령자들이 도움을 요청할 곳이 없다
- 이웃간의 ‘정’이라는 한국 특유의 문화 퇴보

기대효과
1. 현대판 품앗이의 부활으로 이웃간의 정 증진
2. 최종적으로 지역 커뮤니티 또한 형성하여 모든 세대가 이용할 수 있는 커뮤니티 제공
3. 고령자의 디지털 격차 감소
